## Selected work

### [MentorOS](https://github.com/Simon-byte-png/mentoros-app)

A multi-agent intellectual dialogue app with 12 mentor personas, streaming responses, and guided debate flows.

### [Silicon Valley Info Radar](https://github.com/Simon-byte-png/info-radar)

An information workflow that collects high-signal podcasts and RSS feeds, transcribes audio, and turns them into readable Chinese briefs.

### [World Book Atlas](https://github.com/Simon-byte-png/world-book-atlas) · [Live demo](https://simon-byte-png.github.io/world-book-atlas/)

An interactive 3D globe and literary timeline for exploring 154 classic books across 48 countries.

### [Weekly Report System](https://github.com/Simon-byte-png/weekly-report-system)

A Python CLI that aggregates RSS, Atom, sitemaps, and manual sources, then filters, deduplicates, and renders an AI-industry weekly report.

### [Jinggangshan Decision Simulation](https://github.com/Simon-byte-png/jinggangshan-demo)

A classroom-ready historical decision game built with Next.js, with six constrained choices and explainable outcomes.
