I like turning rough ideas into runnable prototypes: classroom demos, AI workflow tools, and systems for thinking with long-term context.

## Featured Projects

### MentorOS

A long-memory, multi-agent intellectual dialogue system. It explores a quieter AI conversation interface with cognitive agents, background council discussion, memory candidates, and decision memo output.

- Tech: Next.js, TypeScript, Tailwind CSS, pnpm workspaces, Turborepo, Supabase-ready packages
- Repo: https://github.com/Simon-byte-png/MentorOS

### Jinggangshan Demo

A classroom-ready Chinese history simulation for《中国近现代史纲要》. Players act as "小徐", a political commissar in the Autumn Harvest Uprising forces, and make six constrained historical decisions.

- Tech: Next.js App Router, TypeScript, Tailwind CSS
- Demo: https://jinggangshan-demo-d9ds5n03b2d27f-1403764965.tcloudbaseapp.com
- Repo: https://github.com/Simon-byte-png/jinggangshan-demo

### Weekly Report System

An AI weekly report CLI / Skill prototype. It aggregates RSS, sitemap and manual sources, then filters, deduplicates and renders a Markdown report.

- Tech: Python standard library, GitHub Actions
- Repo: https://github.com/Simon-byte-png/weekly-report-system
